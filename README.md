# C++ Programming Assignment – 4

## 📌 Description
This repository contains solutions to **Assignment 4 of C++ Programming**.  
The programs are based on **conditional operators, loops, arrays, dynamic memory allocation, and menu-driven logic**.

All programs are written in **C++ (.cpp)** and compiled using **g++ (GCC compiler)**.

---

# 🛠 Concepts Used
- Conditional operator (`?:`)
- If-else statements
- Switch case (menu driven programs)
- Loops (`for`, `while`)
- Arrays (static and dynamic)
- Pointer notation
- Dynamic memory allocation (`new`, `delete`)
- Arithmetic and logical operations

# 📂 List of Programs

### Q1  
Scan 3 integers from user and find the **largest value using conditional operator**.

### Q2

- 1 → Marathi
